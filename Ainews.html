— artificial news, a post-reader —

With this project I combined Artificial Intelligence and natural selection in an phone application. Natural selection in various media does a lot of the intelligent design for us and so we’re moving into the post-intelligent design age and that’s going to be a further evolutionary Darwinian process. You can never predict where evolution is going to take you, because evolution is the amplification of accidents.
The application is collecting news headlines from the internet and by doing so it is learning itself to create new, well written, headlines (based on the ones that were given). Sometimes the outcome is complete gibberish but now and then an “accident” happens and the headline makes sense. The video above is a screen recording of how the application works. By using a tensorflow and python code the application recieves new input.
